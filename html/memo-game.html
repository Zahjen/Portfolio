Memory Game

Description

Envie de se divertir ? Ou tout simplement envie de se remémorer des souvenirs d'enfance ? C'est dans cette optique qu'a été conçu le site Memory qame. Comme son nom l'indique, il s'agit d'un jeu de mémoire, où le but est de retrouver chaque paire cachée, tout en essayant de faire le moins d'essai possible. Il est possible de choisir la taille de la grille parmi 16, 24, 32, 40, 48, 56, 64 cases, pour que la grille soit ensuite générée aléatoirement selon la taille souhaitée.
